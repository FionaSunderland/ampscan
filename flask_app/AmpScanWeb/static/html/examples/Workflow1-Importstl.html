Import stl¶

AmpScan is designed to work on binary stl files. The most basic usage involves importing an stl file into an AmpObject. An AmpObject is the most fundamental part the AmpScan package - it incorporates all the mesh manipulation tools and applies them to the input stl. The basic usage first requires the AmpObject to be imported:

In [1]:

from AmpScan.core import AmpObject

The stl file should be provided as a string containing either the absolute or relative path. Here we use a binary stl sphere saved in the current working directory as sample_stl_sphere_BIN.stl

In [3]:

fh = "sample_stl_sphere_BIN.stl"
amp = AmpObject(fh)
